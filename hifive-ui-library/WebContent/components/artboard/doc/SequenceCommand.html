Class: SequenceCommand

SequenceCommand

new SequenceCommand

複数のCommandを一つのコマンドとして扱うコマンド

SequenceCommandは複数のコマンドを包括するためのコマンドです。

execute()及びundo()時に、登録されたコマンド全ての実行及び取り消しを行います。
