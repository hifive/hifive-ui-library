Mixin: ArtStrokeShape

ArtStrokeShape

ストロークを持つ図形についてのプロパティ定義

以下のクラスがArtStrokeShapeのプロパティを持ちます(プロトタイプにmixinしています)

  • ArtPath
  • ArtRect
  • ArtEllipse

Members

strokeColor :String

ストロークの色

このプロパティにはsetterが設定されており、値を変更すると図形に反映されます

CSSカラー形式の文字列を指定します(#f00,rgb(255,0,0) など)

strokeOpacity :Number

ストロークの透明度(0～1)

このプロパティにはsetterが設定されており、値を変更すると図形に反映されます

strokeWidth :Integer

ストロークの幅

このプロパティにはsetterが設定されており、値を変更すると図形に反映されます
