データグリッド

サンプルはこちら。

説明

  • datagrid は巨大なデータを表形式で表示するためのコンポーネントです。
  • データを配列として一括して渡すこともできますし、都度サーバに問い合わせて取得する方式にすることもできます。
  • 現在のところデータの編集には対応していません。

使用方法

  • h5.ui.components.datagrid 以下にデータグリッドのコントローラを3種類公開しています。
  • ComplexHeaderGridController は固定したいヘッダ部分を HTML で自由に記述できる形式の表です。
  • PagingGridController はページング形式の表です。
  • ScrollGridController は縦スクロール形式の表です。
  • それぞれの詳細な使用方法はサンプルのコードを参考にしてください。
